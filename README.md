# ☕ Cafe Sales Data Analysis & Power BI Dashboard

A data analysis project that explores one month of cafe sales data using **Python (Jupyter Notebook)** and **Power BI**. The project focuses on cleaning raw sales data, performing exploratory data analysis (EDA), and creating an interactive dashboard to generate business insights.

---

## 📌 Project Overview

The objective of this project is to analyze cafe sales data and answer important business questions such as:

- Which products are sold the most?
- What is the total sales revenue?
- Which payment method is most preferred?
- Which salesperson handled the highest number of orders?
- How are sales distributed across different product categories?
- How does daily sales quantity change over time?

The project demonstrates the complete data analysis workflow from raw data to visualization.
